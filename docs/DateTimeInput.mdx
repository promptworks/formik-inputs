# DateTimeInput

Renders an `input[type=datetime-local]` input that will emit ISO-formatted datetime values in UTC on change.

## Basic usage

<Playground>
  <Field initialValue={new Date().toISOString()}>
    {(value, onChange) => <DateTimeInput value={value} onChange={onChange} />}
  </Field>
</Playground>

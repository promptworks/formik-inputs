# DateInput

Renders an `input[type=date]` input that will emit ISO-formatted date values on change.

## Basic usage

<Playground>
  <Field initialValue={new Date().toISOString()}>
    {(value, onChange) => <DateInput value={value} onChange={onChange} />}
  </Field>
</Playground>

# FileInput

Renders an `input[type=file]`. When `multiple` is given, a `FileList` will be emitted on change. Otherwise, either a single `File` or `null` will be emitted.

## Basic usage

<Playground>
  <Field initialValue={null}>
    {(_, onChange) => (
      <FileInput onChange={onChange} />
    )}
  </Field>

  <Field initialValue={[]}>
    {(_, onChange) => (
      <FileInput multiple onChange={onChange} />
    )}
  </Field>
</Playground>

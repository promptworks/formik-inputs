# Select

Renders a select input that will accept a list of options and emit values.

## Basic usage

<Playground>
  <Field initialValue={null}>
    {(value, onChange) => (
      <Select
        value={value}
        onChange={onChange}
        options={['Pizza', 'Spaghetti']}
        placeholder="Choose a food"
      />
    )}
  </Field>

  <Field initialValue='Pizza'>
    {(value, onChange) => (
      <Select
        value={value}
        onChange={onChange}
        options={[
          { label: 'Pizza', value: 'Pizza' },
          { label: 'Spaghetti', value: 'Spaghetti' },
          { label: 'Kale', value: 'Kale', disabled: true }
        ]}
      />
    )}
  </Field>
</Playground>

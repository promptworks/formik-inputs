# Input

Renders an `input[type=number]` input that will emit integer values on change. If the value cannot be parsed as an integer, `null` will be emitted.

## Basic usage

<Playground>
  <Field initialValue={5}>
    {(value, onChange) => <IntegerInput value={value} onChange={onChange} />}
  </Field>
</Playground>

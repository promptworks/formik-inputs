# Checkbox

Renders a `checkbox` input that will emit boolean values.

## Basic usage

<Playground>
  {() => {
    const [value, setValue] = React.useState(true);

    return (
      <div>
        <Checkbox value={value} onChange={setValue} />{" "}
        <pre><code>VALUE: {JSON.stringify(value)}</code></pre>
      </div>
    );

}}

</Playground>
